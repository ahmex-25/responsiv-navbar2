HTML

HTML هي اختصار لـ "HyperText Markup Language" وهي لغة ترميز تُستخدم لبناء هياكل صفحات الويب. يتم استخدام عناصر HTML الموجودة بين علامات الفتح والإغلاق لتحديد هيكل الصفحة وعرض المحتوى بشكل منظم، ويمكن أيضًا إضافة الروابط والصور والفيديوهات والنصوص التوضيحية إليها.

CSS

CSS هي اختصار لـ "Cascading Style Sheets" وهي لغة تنسيق تُستخدم لتعديل شكل وتصميم صفحات الويب المكتوبة بلغة HTML. تُمكن CSS المطورين من تحديد الألوان والخطوط وتنسيق النصوص وتحديد الهوامش والتوازن والتنصيف لتحسين جاذبية الموقع وتجربة المستخدم العامة.

PHP

PHP هي لغة برمجة مفتوحة المصدر وهي عبارة عن لغة سينتكسية تستخدم أساسًا لتطوير تطبيقات الويب والمواقع الديناميكية. يقوم محرك PHP بتحليل الشيفرة على الخادم وتنفيذها، ثم يتم إرسال النتيجة النهائية إلى المتصفح الذي يتصفح الموقع.

jQuery

jQuery هي مكتبة JavaScript مفتوحة المصدر ومتعددة المهام تُستخدم لتبسيط كتابة البرمجيات وتنفيذ عمليات DOM (Document Object Model) والتعامل مع الأحداث والتلاعب بالعناصر الموجودة في صفحات الويب. يساعد jQuery المطورين على كتابة أكواد قصيرة وسهلة التفاعل مع مستندات HTML وCSS، وتسهيل تطبيق التأثيرات والتحقق من البيانات المدخلة من المستخدم.
